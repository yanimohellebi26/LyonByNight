# Lyon Night Guide

## Tagline
Interactive nightlife guide for Lyon with AI chatbot, interactive map, and venue comparator.

## Description
A full-stack web application for discovering Lyon's nightlife — bars, clubs, concerts, and events. Features an AI-powered chatbot for personalized recommendations, an interactive Mapbox map with clustering, advanced filters (type, neighborhood, ambiance, price), and a side-by-side venue comparator. Built with modern web technologies and deployed on Vercel.

## Motivation
Solving a real local need — there's no good single tool for discovering Lyon's nightlife scene. This combines curated data, geospatial search, and AI-powered recommendations into one polished experience.

## Tech Stack
- **Frontend**: Next.js 16, TypeScript, Tailwind CSS v4, shadcn/ui, Framer Motion
- **Backend**: Vercel AI SDK, LangChain.js, Cheerio (scraping)
- **Database**: Supabase (PostgreSQL + PostGIS + pgvector)
- **AI/ML**: OpenAI GPT-4o, RAG pipeline
- **Maps**: Mapbox GL JS
- **APIs**: Yelp Fusion, Foursquare
- **Media**: Cloudinary
- **DevOps**: Vercel, i18n (FR/EN)

## Key Features
- Interactive Mapbox map with clustering, dark mode, and geolocation
- RAG-powered AI chatbot for personalized nightlife recommendations
- Advanced filters (type, neighborhood, distance, rating, ambiance, price)
- Side-by-side venue comparator (up to 4 venues)
- Event scraping and agenda display
- Internationalization (French/English) and PWA-ready

## Skills Demonstrated
- Full-stack Next.js development with App Router
- Geospatial queries with PostGIS
- RAG pipeline design and implementation
- Third-party API integration (Yelp, Foursquare, Mapbox)
- PWA architecture and i18n
